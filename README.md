# Simple Interest Calculator

## 📄 Description

This is a basic shell script project to calculate **Simple Interest** based on user input. It prompts the user to enter the **Principal**, **Rate of Interest**, and **Time**, then calculates and displays the result. The project demonstrates the use of GitHub workflow, version control with Git, and collaboration best practices.
